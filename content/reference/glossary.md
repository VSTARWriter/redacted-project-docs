# Glossary

Common terms used in the application.

## A

**Active Company**
: The company associated with your user account.

**Active District**
: The geographic district you're currently working in.

**Active Location**
: The specific location within your district.

## B

**Bit**
: A drill bit tracked in the inventory system.

**Bit Details**
: The detailed information page for a specific bit.

**Bit Order**
: A request for bits from inventory.

**Bit Status**
: The current state of a bit (e.g., available, in use, locked).

## C

**Consignment**
: Placing a bit with a customer under consignment terms.

## D

**Delivery Ticket**
: Documentation for delivering bits to a customer.

**District**
: A geographic region for organizing inventory and operations.

## E

**eBilling Workbench (EBW)**
: The main view for managing service receipts and billing.

**eTransfer**
: Electronic transfer of bits between locations or users.

## F

**Find Bits**
: The advanced search view for locating specific bits.

## I

**Inventory Management**
: The home view for managing bit inventory (Inventory Manager role).

## L

**Locked Serial**
: A bit that has been locked to prevent transfers.

**Look Back Days**
: The number of historical days to include in a search.

## M

**My Bits**
: The home view showing bits assigned to you (Field Sales role).

## N

**Nav History**
: Historical transaction records from the navigation system.

## P

**Pending**
: Items awaiting action or completion.

## S

**Serial Number**
: The unique identifier for each bit.

**Service Receipt (SR)**
: Documentation of service performed with a bit.

**Stockpoint**
: A specific inventory storage location.

## T

**Ticket Line**
: A line item within a service receipt or delivery ticket.

**Transactions**
: The view showing bit orders and inventory movements.
