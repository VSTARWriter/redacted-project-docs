# Inventory Management View

The Inventory Management view is the home page for the Inventory Manager role, providing tools to manage bit inventory and stock levels.

!!! info "Role Availability"
    This view is available to: Inventory Manager

## Overview

![Inventory Management Placeholder](../assets/images/placeholder-table-view.svg)

The Inventory Management view displays all bits in your inventory with their current status, location, and details.

## Quick Action Buttons

| Button | Function |
|--------|----------|
| **Transactions** | Go to Transactions view |
| **eTransfer Tickets** | Go to eTransfer Tickets view |
| **Find Bits** | Go to Find Bits view |

## Scanning Features

### Scan Bit Serial No

Add bits to the pending queue:

- **Desktop:** Enter serial number manually
- **Mobile/iPad:** Scan the barcode

## Table Features

### See Locked Serials Toggle

Toggle **See Locked Serials** on/off to show or hide locked bits in the inventory list.

### Global Search

Use the search box above the table to search across all columns:

- Serial Number
- Item No
- Status
- And more

### Sorting and Filtering

**To sort:**
1. Click to the left of the funnel icon in a column header
2. Sort arrows appear
3. Click up/down arrow to sort

**To filter:**
1. Click the funnel icon in a column header
2. Select filter options from the popup
3. Filters remain until cleared

**To clear:**
Click **Clear filter-sort** to reset all sorting and filtering.

## Viewing Bit Details

Click any row in the table to open the **Bit Details** page for that bit.
