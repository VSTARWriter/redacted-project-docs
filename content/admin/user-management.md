# User Management

Add and edit user accounts in the system.

!!! info "Role Requirement"
    This feature requires the **IT Support** role.

## Overview

The ADMIN: Add/Edit Users view is the home page for IT Support users.

![User Management Placeholder](../assets/images/placeholder-table-view.svg)

## Adding Users

1. Log in with IT Support role.
2. Click the **Add User** button.
3. Enter the user's identification number.
4. Enter the user's contact information.
5. Assign a role to the user.
6. Assign District/Location access to the user.
7. Click the **Save** button to confirm the new user record.

## Editing Users
1. Use the search function to locate the user.
2. Select the user record.
3. Update any necessary fields.
2. Click the **Save** button to confirm your changes.
