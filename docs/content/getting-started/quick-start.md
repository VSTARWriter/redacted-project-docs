# Quick Start

Get up and running with Project Documentation in minutes.

## Step 1: Install the Application

Choose your platform:

- **Desktop (Windows):** [Desktop Installation Guide](../installation/desktop.md)
- **Mobile/iPad:** [Mobile Installation Guide](../installation/mobile.md)

## Step 2: Log In

1. Open the application
2. Enter your network email address
3. Enter your password
4. Complete multi-factor authentication

## Step 3: Configure Your Profile

On first login, set up your profile:

1. Navigate to **Profile** from the menu
2. Select your **Active District**
3. Select your **Active Location**
4. Select your **Stockpoint**
5. Click **OK** to save

## Step 4: Start Working

You'll be taken to your role's home page:

| Role | Home Page |
|------|-----------|
| Inventory Manager | Inventory Management |
| Field Sales | My Bits |
| Billing | eBilling Workbench |
| Rep Manager | Add/Edit Customer Rep |
| Viewer | Find Bits (Lite) |
| Account Rep | eBilling Workbench |
| Finance | eBilling Workbench |
